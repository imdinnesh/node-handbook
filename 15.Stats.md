# Node.js File Statistics

Every file comes with a set of details that we can inspect using Node.js. Specifically, we use the `stat()` method provided by the `fs` module to retrieve these details.

## 1. Using the `fs.stat()` Method

You call `fs.stat()` by passing a file path. Once Node.js retrieves the file details, it executes a callback function with two parameters: an error message and the file stats object.

### Callback Pattern (Asynchronous)
```javascript
const fs = require('node:fs');

fs.stat('/Users/joe/test.txt', (err, stats) => {
  if (err) {
    console.error(err);
    return;
  }
  // Access file details via the `stats` object
});
```

### Synchronous Pattern
Node.js also provides a synchronous version that blocks the execution thread until the file stats are ready:

```javascript
const fs = require('node:fs');

try {
  const stats = fs.statSync('/Users/joe/test.txt');
} catch (err) {
  console.error(err);
}
```

> [!TIP]
> Use the synchronous method sparingly in production environments, as it blocks the event loop.

---

## 2. The `Stats` Object

The `stats` variable contains a wealth of information about the file. Here are the most common methods and properties you'll use:

| Method / Property | Description |
| :--- | :--- |
| `stats.isFile()` | Returns `true` if it's a regular file. |
| `stats.isDirectory()` | Returns `true` if it's a directory. |
| `stats.isSymbolicLink()` | Returns `true` if it's a symbolic link. |
| `stats.size` | The file size in bytes (e.g., `1024000` = 1MB). |

### Example Usage
```javascript
const fs = require('node:fs');

fs.stat('/Users/joe/test.txt', (err, stats) => {
  if (err) {
    console.error(err);
    return;
  }

  console.log(stats.isFile());           // true
  console.log(stats.isDirectory());      // false
  console.log(stats.isSymbolicLink());   // false
  console.log(stats.size);               // 1024000 (1MB)
});
```

---

## 3. Promise-based Approach

If you prefer modern `async/await` syntax, you can use the `fs/promises` module:

```javascript
const fs = require('node:fs/promises');

async function getFileStats() {
  try {
    const stats = await fs.stat('/Users/joe/test.txt');
    
    console.log(`Is File: ${stats.isFile()}`);
    console.log(`Size: ${stats.size} bytes`);
  } catch (err) {
    console.error(err);
  }
}

getFileStats();
```

> [!NOTE]
> There are many other advanced methods available in the `stats` object, but the ones listed above represent the bulk of day-to-day programming needs.
