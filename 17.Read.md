# Reading Files with Node.js

Node.js provides several ways to read files, ranging from simple one-off reads to high-performance streaming for large datasets.

## 1. Using Callbacks (Asynchronous)
The simplest way to read a file in Node.js is to use the `fs.readFile()` method. It passes the file path, encoding, and a callback function that handles the data or error.

```javascript
const fs = require('node:fs');

fs.readFile('/Users/joe/test.txt', 'utf8', (err, data) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log(data);
});
```

## 2. Using the Synchronous API
Alternatively, you can use the synchronous version `fs.readFileSync()`. This blocks execution until the file is read.

```javascript
const fs = require('node:fs');

try {
  const data = fs.readFileSync('/Users/joe/test.txt', 'utf8');
  console.log(data);
} catch (err) {
  console.error(err);
}
```

## 3. Using Promises (Asynchronous)
The `fs/promises` module offers a promise-based `readFile()` method, which is ideal for modern `async/await` syntax.

```javascript
const fs = require('node:fs/promises');

async function example() {
  try {
    const data = await fs.readFile('/Users/joe/test.txt', { encoding: 'utf8' });
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

example();
```

> [!WARNING]
> **Memory Consumption**
> All three methods above (`fs.readFile()`, `fs.readFileSync()`, and `fsPromises.readFile()`) read the **full content** of the file into memory before returning the data.
>
> For large files, this can significantly impact memory usage and program performance.

## 4. Reading with Streams
For large files, a better option is to read the content using **Streams**. This allows you to process the file piece by piece (chunks) without loading everything into memory at once.

```javascript
import fs from 'fs';
import { pipeline } from 'node:stream/promises';
import path from 'path';

const fileUrl = 'https://www.gutenberg.org/files/2701/2701-0.txt';
const outputFilePath = path.join(process.cwd(), 'moby.md');

async function downloadFile(url, outputPath) {
  const response = await fetch(url);
  
  if (!response.ok || !response.body) {
    // consuming the response body is mandatory: https://undici.nodejs.org/#/?id=garbage-collection
    await response.body?.cancel();
    throw new Error(`Failed to fetch ${url}. Status: ${response.status}`);
  }

  const fileStream = fs.createWriteStream(outputPath);
  console.log(`Downloading file from ${url} to ${outputPath}`);
  
  await pipeline(response.body, fileStream);
  console.log('File downloaded successfully');
}

async function readFile(filePath) {
  const readStream = fs.createReadStream(filePath, { encoding: 'utf8' });

  try {
    for await (const chunk of readStream) {
      console.log('--- File chunk start ---');
      console.log(chunk);
      console.log('--- File chunk end ---');
    }
    console.log('Finished reading the file.');
  } catch (error) {
    console.error(`Error reading file: ${error.message}`);
  }
}

try {
  await downloadFile(fileUrl, outputFilePath);
  await readFile(outputFilePath);
} catch (error) {
  console.error(`Error: ${error.message}`);
}
```
